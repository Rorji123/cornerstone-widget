FAQ

CSS-only accordion using semantic <details>.

Can I paste this into GitHub Pages directly?

Yes — add index.html and styles.css to the root of your repo, then enable GitHub Pages in repo settings.

Does this require images or external libraries?

Nope. All visuals are gradients, shapes, and CSS effects. It’s purposely self-contained.

Is it mobile-friendly?

Yep — responsive grids, fluid type with clamp(), and a CSS-only mobile menu.

Can you tailor it to my brand / colours / sections?

Definitely. If you tell me your brand colours, logo text, and what sections you want, I’ll re-theme it and add content (e.g., portfolio, product, HR dashboard, etc.).
